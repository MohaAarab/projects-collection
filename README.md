A curated set of automation workflows, scripts, and tools I built for clients across different industries. All projects are anonymized and reusable for anyone facing similar challenges.

## Prefix System

| Prefix | Category                    | Example          |
|--------|-----------------------------|------------------|
| AUT    | Automation                  | AUT-001, AUT-002 |

| MAR    | Marketing/ Scraping         | MAR-001          |
| DEV    | Dev Tools                   | DEV-001          |

Followed by a 3-digit serial ID (e.g., `AUT-001`).

## How to Use These Projects

1. **Browse folders** to find the project you need
2. **Read each project's README.md** for setup instructions
3. **Import workflow files** (e.g., `workflow.json` for n8n)
4. **Configure environment variables** using `.env.example` templates
5. **Never commit real credentials**—use your own `.env` files

## Security Notes

- No real API keys, passwords, or client data included
- All configs use example/dummy values
- Replace with your credentials before deployment
